ULTRASPEC target list

The target list below is generated dynamically from the current archive of ULTRASPEC runs. The list is used by Python scripts to match the strings listed on the right-hand side (past target names entered at the telescope) to produce the ID, RA and Dec listed on the left-hand side. If you wish to observe a target in the list, please use the name (or one of the names) used in the past as it greatly helps maintenance of the archive. Tildes (~) indicate spaces; please replace them with spaces if you use a given name.
