ULTRASPEC run search page.

This page allows position-based searches for through the ULTRASPEC runs and also contains the complete list of objects we have observed with ULTRASPEC. The searches return all runs which satisfy the selection criteria below, which are currently based upon distance on the sky and exposure time. You can either enter the position explicitly or click on one of the objects in the target list. The search results are displayed in a new page.
